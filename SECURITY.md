# Security Policy

## Supported Versions

As this is a project in its early stages of development, only the latest version is actively supported. Please ensure you are using the most recent commit from the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

I take all security bugs in GoTune seriously. Thank you for improving the security of this software. I appreciate your efforts and responsible disclosure and will make every effort to acknowledge your contributions.

If you believe you have found a security vulnerability, please report it privately through [GitHub's security advisory system](https://github.com/tejashwikalptaru/gotune/security/advisories/new). I will do my best to respond to your report within 48 hours.

Please include the following information in your report:
* A description of the vulnerability and its potential impact.
* Steps to reproduce the vulnerability.
* Any related information, such as screenshots or code snippets.

I will review your report and if the vulnerability is accepted, I will work on a fix and release a new version as soon as possible. You will be credited for the discovery.
